在第一篇文章之后，我继续深入学习 Astro 框架。

## Astro 的优势

Astro 是一个现代化的静态站点生成器，具有以下特点：

- 🚀 极快的页面加载速度
- 📦 零 JavaScript 默认输出
- 🎨 支持多种 UI 框架
- 📝 内置 Markdown 支持

## 我的学习心得

通过实践，我发现 Astro 非常适合构建内容驱动的网站，比如博客、文档站点等。

它的组件化开发方式让代码组织更加清晰，而且性能表现出色。
